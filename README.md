# ⌨️安装

### 在项目的pom.xml中添加如下坐标

```xml
<dependency>
    <groupId>cn.hutool</groupId>
    <artifactId>hutool-all</artifactId>
    <version>5.7.13</version>
</dependency>
```





# 🔥使用方法

### 1.直接下载编译后的jar包运行

##### 点击<a href="1">这里</a>下载最新的编译jar包

##### 在控制台中运行

```shell
java -jar DouyinSpider.jar
```
